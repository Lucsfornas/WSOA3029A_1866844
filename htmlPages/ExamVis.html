This is my Exam interactive Data visualisation made using d3.js libraries displaying the population in each province in South Africa by selecting each button the graph will display the Covid-19 confirmed cases and deaths in each province. This Information was scourced from: statssa.gov.za and covid19.org


The following numbers represent these provinces:


1: Gauteng Province (GP)

2: KwaZulu-Natal (KZN)

3: Western Cape (WC)

4: Eastern Cape (EC)

5: Limpopo (LIM)

6: Mpumalanga (MPUM)

7: North-West (NW)

8: Free State (FS)

9: Northern Cape (NC)
